# Relembrando comando GIT/GITHUB

## Configurações
Adiciona o nome do desenvolvedor:
```bash
git config --global user.name "Lucas Andriotti Ciamaricone"
```
Adiciona o e-mail do desenvolvedor:
```bash
git config --global user.email lucasandriotticiamaricone.com
```

## Comando Git
INIT inicia o repositório 
ADD adiciona tudo para o computador atual
STATUS mostra o status do nosso git
COMMIT salva uma parte do código
